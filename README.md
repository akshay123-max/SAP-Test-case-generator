# SAP Test Case Generator (Free Edition)
**Features**
- Free & cloud-based (no API keys)
- Accepts any document format (PDF, DOCX, XLSX, TXT, CSV, etc.)
- Smart context extraction and test case generation
- Uses Hugging Face models (Gemma + MiniLM)
- Excel export and preview dashboard
